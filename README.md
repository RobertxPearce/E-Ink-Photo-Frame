# E-Ink-Photo-Frame
Photo frame using Pimoroni's Inky Impression 7.3 e-ink display.

## How to Use
1. **SSH into Pi**
2. **Move to Directory**
   ```bash
   cd /Desktop/E-Ink-Photo-Frame/Code
   ```
4. **Run the Program**
   ```bash
   python3 image_static.py
   ```
   or
   ```bash
   python3 image_loop.py
   ```
